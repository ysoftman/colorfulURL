# Colorful URL (chrome extension)

## 설치

- chrome://extensions/ -> 개발자 모드 활성화 -> 압축해제된 확장 프로그램을 로드합니다.(LOAD UNPACKED) -> 프로젝트 디렉토리 선택

## 디버깅

- 코드 수정 후 chrome://extensions/ 에서 '업데이트' 하면 바로 적용된다.
- 앱의 '백그라운드 페이지' 에서 콘솔 로그를 확인한다.
